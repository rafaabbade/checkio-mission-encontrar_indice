Enunciado

Sofia, uma bibliotecária digital, está organizando uma grande lista de títulos de livros. Ela precisa de uma ferramenta que a ajude a encontrar a posição de um livro específico na lista para poder acessar rapidamente as informações detalhadas sobre ele. Escreva uma função chamada encontrar_indice que receba uma lista de strings (títulos de livros) e uma string (título de um livro). A função deve retornar o índice do título na lista. Se o título não estiver na lista, retorne -1.

Entrada: Uma lista de strings e uma string.

Saída: Um número inteiro representando o índice do título na lista ou -1 se o título não estiver na lista.
